# CoExtracter
Comment Extracter with python

# Usage

- for help :

```
  python3.5 main.py --help
```


```
  python3.5 main.py -h
```

- keywords

```
  -i , --ifile = this means name of your input file
  -o , --ofile = this means name of your output file
  -s , --sword = this means word of your want to search and extract
```
